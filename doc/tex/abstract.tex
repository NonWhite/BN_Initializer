\begin{abstract}
	Este artigo propõe uma versão do algoritmo de aprendizado de redes bayesianas que usa o score BIC e uma busca no espaço de ordenamentos. A única modificação do algoritmo é feita no passo de inicialização de uma permutação dos atributos. Um conjunto de experimentos mostra que fazendo esta modificação no algoritmo obtemos redes melhores para os conjuntos de dados.
\end{abstract}
